# GIT

## COMANDOS BÁSICOS

1) Trazendo possíveis mudanças do GitHub:
git pull

2) Adicionar arquivos para commit:
git add .

3) Criar versionamento (commit):
git commit -m "comandos básicos do git."

4) Enviar alterações para GitHub:
git push


## Branch

Mostrar branches no computador:
git branch
